# **Pharmacy Management System**

This Project is a Pharmacy Management System with CRUD operations the technologies used are:

## **Front-end tools and technologies used are:**
- Angular 16
- TypeScript
- NodeJS and NPM
- VS Code IDE
- Angular CLI
- Bootstrap 5

## **Back-end tools and technologies used are:**
- Spring Boot 4
- Hibernate
- Maven
- JDK 17
- Embedded Tomcat
- MySQL Database
